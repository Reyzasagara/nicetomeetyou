## Image Specifications

- **Format**: JPG or PNG
- **Recommended size**: 1200x900px (4:3 ratio) or 1600x900px (16:9 ratio)
- **Maximum file size**: 500KB per image (optimize for web)
- **Content**: Screenshots of your work, diagrams, dashboards, documentation, or project photos

## Image Gallery Features

✅ **Horizontal scroll**: Swipe left/right to see all images
✅ **Hover effect**: Images scale up and glow on hover
✅ **Responsive**: Adjusts size on mobile/tablet
✅ **Custom scrollbar**: Cyan-colored scrollbar matches site theme
✅ **Clickable**: Each image is clickable (can add lightbox later if needed)

## Tips for Good Documentation Images

1. **Blur sensitive data**: Use image editing to blur company-specific information
2. **High contrast**: Make sure text and details are readable
3. **Professional**: Use screenshots from actual work, not stock photos
4. **Variety**: Mix different types of documentation (code, UI, diagrams, reports)
5. **Quality**: Use clear, well-lit photos if taking pictures of physical documents
